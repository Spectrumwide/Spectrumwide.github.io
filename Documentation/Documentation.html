SASS Architecture üèóÔ∏è

The project uses a modular SASS structure for better organization and maintenance:

  • Partials Organization

    • _variables.scss - Global variables and configurations
    • _mixins.scss - Reusable mixins and functions
    • _effects.scss - Animation and interactive effects
    • Component-specific partials in respective folders

  • Build Process

    • Gulp-based SASS compilation
    • Automatic file watching and rebuilding
    • CSS optimization and minification
