This project is a simple HTML application designed for Pokemon enthusiasts. It allows users to retrieve and display Pokemon sprite images using the PokeAPI. The application features a user-friendly interface where users can input the name or ID of a Pokemon to fetch its sprite. It utilizes modern web technologies such as HTML, CSS, and JavaScript to create an interactive experience. The PokeAPI is integrated to fetch real-time data, ensuring accurate and up-to-date Pokemon information. The application is lightweight and responsive, making it accessible across various devices and screen sizes. Error handling is implemented to provide feedback if the user enters an invalid Pokemon name or ID. This project demonstrates the use of API calls, DOM manipulation, and event handling in JavaScript.
