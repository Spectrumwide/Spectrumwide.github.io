Strings



Strings is a Conceptual App Design that represents an online Music Streaming Service App designed using Figma. This Design could be further considered to develop an actaual functioning application in future.


View Figma

Case Study

The Design of this application is loaded with prototyping that provides an immersive user experience similar to that of an actual working application. The App design screens consists of Login Screen for the existing user as well as registration page for the new users to register.It consists of few form fields asking for user data to proceed registration.Then there is forget password screen. Including the initial loading screen, initially there are 3 screens before login.


Then the main landing screen is divided into 4 screens Home screen, Search, Libraries and More. Each screens can be navigated as individual tabs using the bottom navigation. The Navigation bar and music player tab will be sticky and will be present across almost all screens.


Home Screen

Home screen consists of welcome note, recent playlists and favorites and small mediaplayer tab to navigate to main media player screen. The podcast tab can also be implemented as a future enhancement.


Search

This screen is for the user who wants to perform global search of a respective Music Track. This screen consists of a search bar, recent history of music track and artist card.

Library

This screen consists of playlists created by users or the public playlists and artists saved.By clicking on each playlist, the user will be taken to respective playlist screen which displays playlist cover image, shuffle play icon, title and respective data, songs lists. On clicking on each song card, respective song will be played.


The Main player screen will have a like button, more options, album cover image, title of track, and unique user design representing a retro sony walkman. Refer the Screenshots for reference.


More

More options will be for users to set any configurations and the options will be present to configure the app. The options can be related to setting my profile, Language filters, Equaliser, Notifications, Subscriptions, Accessibility etc.



Note: This application is a visual design which is representing a concept and is developed just to get the sight or overview of how the app will look like before its being fully developed.
